# Task-Management

## About the Project

With Task-Management, you can plan your tasks and take notes. Your tasks are saved in localstorage.

- Creating a section
- Creating a task
- Editing a section
- Deleting a section
- Relocating tasks
- Deleting a task

The following technologies are used in the Task-Management application:

- Vue2
- Vue Router
- Vuex
- Vue Hero Icons
- Ant Design Vue
- Vue Draggable

[Demo](https://task-management-smoky.vercel.app/)

### Install Dependencies

```
npm install
```

### Run the Project

```
npm run serve
```
